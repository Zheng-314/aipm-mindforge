# 🚀 AIPM MindForge - AI产品经理智能学习平台

## 📋 项目概述
这是一个专为AI产品经理设计的智能学习平台，整合了知识图谱、智能笔记和AI分析功能。

## 🏗️ 项目结构
\\\
D:\AIPM_project/
├── AIPM/                    # Spring Boot后端
├── mindforge-frontend/      # Vue.js前端
├── uploads/                 # 用户上传文件（不纳入版本控制）
└── .gitignore              # Git忽略配置
\\\

## 🛠️ 技术栈
- **后端**: Spring Boot 3 + MySQL + JPA + JWT
- **前端**: Vue 3 + TypeScript + Element Plus + Vite
- **AI集成**: DeepSeek API

## 🚦 快速开始
1. 配置数据库
2. 启动后端服务
3. 启动前端开发服务器
